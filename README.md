# CIBI Platform

Full-stack CIBI Bible College LMS starter.

## Features

- Public website
- Public book library with Stellar purchase links
- Student registration
- Paystack payment initialization and verification
- Manual bank transfer fallback
- Student portal locked until admin approval
- Student WhatsApp group only inside active student portal
- Admin dashboard for books, courses, lessons, homepage slides, announcements, live classes, settings, students and approvals
